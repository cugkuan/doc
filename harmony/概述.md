#  相关的工具

## HDC 的配置

hdc 是IP连接手机的服务

鸿蒙关于Hdc 的配置，太简单，按照企配置并不能完成。下面是完整的配置

- 在 .zshrc文件中，输入下面的内容
```shell
export PATH="$PATH:/Users/kuan/Library/Huawei/Sdk"
export PATH="$PATH:/Users/kuan/Library/Huawei/Sdk/openharmony/9/toolchains"
export HDC_SERVER_PORT=7035
```

主要是找到 toolchains 的路径

- shell终端中输入

```
source ~/.zshrc
```

- shell终端中输入
```
hdc -h
```
验证 hdc 是否生效了。
